# Badan Warisan Malaysia App - Navigation Guide

## All Available Screens

### Main Screens (Bottom Navigation)
1. **Home** - Main dashboard with campaigns, quick actions, and community features
2. **Donate** - Donation interface (placeholder)
3. **Events** - List of heritage events and programmes
4. **Profile** - User profile with stats and menu options

### From Home Screen
- **Membership** - Click "Membership" button or access from profile
- **Leaderboard** - Click "Leaderboard" icon in icon menu
- **AI Assistant** - Click "AI Help" icon in icon menu
- **Community Wall** - Click "View all community messages" link
- **Heritage Passport** - Access via floating Quick Access button (gold menu icon)
- **My Journal** - Access via floating Quick Access button

### From Profile Screen
- **My Membership Card** - View digital membership card with QR code
- **Heritage Passport** - Track heritage site visits and collect stamps
- **Donation History** - View past donations with downloadable receipts
- **My Journal** - Document event experiences with photos and notes
- **Settings** - App settings (placeholder)

### From Events Screen
- **Event Details** - Click "Book Now" on any event card

## Quick Access Features

The **floating golden menu button** (bottom right on home screen) provides quick access to:
- Heritage Passport - Gamification feature to collect site visit stamps
- My Journal - Personal heritage experience documentation
- Community Wall - Read supporter messages and testimonials

## Color Scheme
- Heritage Green: `#0A402F`
- Cream Background: `#FEFDF5`
- Heritage Gold: `#B8860B`
- Charcoal Text: `#333333`
- White Cards: `#FFFFFF`

## Typography
- Headings use **Lora** font (serif)
- Body text uses default system font

## Key Features

### 1. Heritage Passport (Gamification)
- Track visits to 6 heritage sites
- Unlock sites by attending BWM events
- Progress tracking with visual stamps
- Reward: 10% off membership renewal when complete

### 2. My Journal
- Document event experiences
- Upload photos and write reflections
- Community engagement with likes and comments
- Fulfill SULAM service learning reflection objectives

### 3. Community Wall
- Public messages from donors and supporters
- Social proof and community building
- Encourages donations and engagement

### 4. AI Heritage Assistant
- Interactive chatbot for BWM information
- Answers questions about events, membership, heritage
- Suggested prompts for quick queries

### 5. Leaderboard
- Top Donors and Top Volunteers tabs
- Gold/Silver/Bronze rankings for top 3
- Encourages friendly competition
- Privacy note about anonymous donations
